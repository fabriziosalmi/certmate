Choose where to store your SSL certificates. Local filesystem is the default and most compatible option.

Local Filesystem Storage

Certificates are stored locally in the specified directory with secure file permissions.

Azure Key Vault Configuration

Use Certificate object or Both for native consumption from App Service, Application Gateway, Front Door, API Management or AKS Ingress. The Service Principal needs Certificates Get/List/Import/Delete in addition to Secrets Get/List.

Imports a Certificate object for every domain that already lives in this vault as Secrets but does not yet have one. Skips entries already imported.
